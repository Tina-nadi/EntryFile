    3-APIs In PHP

    They are divided into several types based on their architecture and mode of operation , RESTful API (Representational State of Resource),GraphQL API , SOAP API (Simple Object Access Protocol), gRPC API ,WebSocket API.

    - Request and Response
    An API request is made by a client application to a server to retrieve, create, update, or delete data. The request typically includes the following components:
  • HTTP Method(common method : GET , POST , PUT , DELETE)
  • URL (Endpoint)
  • Headers

    • An API response is the serverâ€™s reply to an API request :
  • Status Code (like 200 means ok ..)
  • Headers
  • Body
    • - End Point in API? In API, an End Point (or Endpoint) is a unique address that identifies a specific service or resource on a server. End Points allow clients to interact with the server and perform various operations.

    Examples of End Points:
    GET /users: Retrieve a list of users
    POST /users: Create a new user
    GET /users/123: Retrieve information about a user with ID 123
    PUT /users/123: Update information about a user with ID 123
    DELETE /users/123: Delete a user with ID 123

    -Format Files

        const xhr = new XMLHttpRequest();

          }
        };
        xhr.send();
